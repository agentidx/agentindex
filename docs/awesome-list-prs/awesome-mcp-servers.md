## PR body
Nerq Gateway provides trust verification for MCP servers. It runs preflight checks against 25,000+ indexed servers, returning trust scores, license info, and maintenance signals before your agent connects. Includes task-to-tool resolution so agents can describe what they need and get matched to verified servers.

- Homepage: https://nerq.ai/gateway
- npm: https://www.npmjs.com/package/nerq-gateway
- Install: `npm i -g nerq-gateway`
