## PR title
Add nerq — trust verification SDK for AI agents

## PR body
nerq is a lightweight Python SDK for trust verification of AI agents and tools. It provides preflight checks (license, security, maintenance signals) and task-to-tool resolution so LangChain pipelines can verify dependencies before execution.

- PyPI: https://pypi.org/project/nerq/
- Install: `pip install nerq`
- Homepage: https://nerq.ai
