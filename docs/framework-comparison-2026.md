## Methodology

Framework association is determined from the `frameworks` array in the Nerq agent database. An agent is counted under a framework if that framework appears in its declared dependencies, metadata, or detected integrations. Trust scores are the Nerq Trust Score v2 — a composite of security practices, compliance, maintenance activity, community trust, and ecosystem compatibility.
