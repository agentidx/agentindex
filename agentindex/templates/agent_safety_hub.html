How Nerq Safety Scores Work

Every AI tool in the Nerq index receives an automated Trust Score from 0 to 100, calculated from 13+ independent signals across five dimensions: Security (CVEs, dependency vulnerabilities), Maintenance (commit activity, release cadence), Documentation (README quality, API docs), Compliance (license, regulatory alignment), and Community (stars, forks, downloads). Tools scoring 70+ earn Nerq Verified status. Scores update continuously as new data becomes available.
