Adding the Nerq Trust MCP server, which provides trust verification for AI agents and tools.

Tools exposed:
- `check_trust` — Get trust score and grade for any AI agent
- `compare_agents` — Side-by-side comparison of two agents
- `recommend` — Get agent recommendations for a task
- `scan_dependencies` — Bulk trust check for requirements.txt

Backed by nerq.ai's database of 204,000+ indexed agents.
