Methodology

Nerq Trust Score is a composite 0-100 score based on five pillars: Security (30%), Compliance (25%), Maintenance (20%), Popularity (15%), Ecosystem (10%). Agents scoring 60+ are TRUSTED, 35-59 CAUTION, below 35 UNTRUSTED. Data is crawled from npm, PyPI, HuggingFace, GitHub, Docker Hub, and MCP registries.
